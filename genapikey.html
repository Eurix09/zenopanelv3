API Key Management ▼
/key

Get API Keys

Method: GET

Description: Retrieve all API keys for authenticated user

Requires: Authorization token

/add-key

Add API Key

Method: POST

Description: Add a new API key

Requires: Authorization token

{ "apiKey": "string", "expirationDate": "YYYY-MM-DD", "zipCode": "string", "type": "1 Key 1 Dev or 1 Key All Dev" }
/removekey

Remove API Key

Method: POST

Description: Remove an API key

Requires: Authorization token

{ "apiKey": "string" }
/toggle-key-maintenance

Toggle Key Maintenance

Method: POST

Description: Toggle maintenance mode for an API key

Requires: Authorization token

{ "apiKey": "string", "forceDisable": boolean (optional) }
